Uma estrutura de repetição comum em Python é o `for`. Aqui está um exemplo que imprime os números de 1 a 5:

```python
for i in range(1, 6):
    print(i)
```

Nesse código, `range(1, 6)` gera uma sequência de números de 1 a 5, e o `for` percorre cada número, imprimindo-o.

Uma estrutura de tomada de decisão em Python é a `if` statement. Aqui está um exemplo que verifica se um número é positivo, negativo ou zero:

```python
num = int(input("Digite um número: "))

if num > 0:
    print("O número é positivo.")
elif num < 0:
    print("O número é negativo.")
else:
    print("O número é zero.")
```

Nesse código, o `if` verifica se o número é positivo, o `elif` verifica se é negativo e o `else` cobre o caso em que o número é zero.
